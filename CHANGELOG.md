# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-04

### Added
- Initial release of Mathematica to LaTeX converter
- Comprehensive symbol conversion (Greek letters, mathematical symbols)
- Support for subscripts and superscripts
- Table extraction from GridBox structures
- Graphics detection with placeholder generation
- Code block formatting with syntax highlighting
- Multi-file batch processing
- Professional documentation and examples
- MIT License
- Contributing guidelines
- Issue and PR templates

### Features
- Converts .nb files to compilable LaTeX documents
- Preserves both Mathematica code and output
- Automatic section headings and formatting
- Clean, readable LaTeX output
- Extensive symbol mapping
- No external dependencies (Python 3.6+ only)

### Documentation
- Comprehensive README with usage examples
- Example notebooks and output files
- Symbol conversion reference
- Setup instructions for PyPI distribution
